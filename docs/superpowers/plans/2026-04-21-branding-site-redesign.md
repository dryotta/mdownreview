# Branding & Site Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Apply consistent indigo brand identity across the site and app, simplify the "How It Works" layout, add a platform-aware install tab component, and remove all Copilot references.

**Architecture:** Pure HTML/CSS/JS changes to `site/`, CSS + TSX changes to the React app's WelcomeView, and icon regeneration via the Tauri CLI. No new build tooling required.

**Tech Stack:** HTML/CSS/JS (site), React + TypeScript (app), Tauri CLI v2 (`npm run tauri -- icon`) for icon generation.

    function detectPlatform() {

      const pl = (navigator.platform ?? '').toLowerCase();
      if (ua.includes('win')) {
        return (ua.includes('arm') || pl.includes('arm')) ? 'win-arm64' : 'win-x64';
      }
      if (ua.includes('mac')) return 'mac-arm64';
      return null;

    if (detected && PLATFORM_LABELS[detected]) {

    if (detected) {
