# DMG layout assets

`background.png` and `background@2x.png` are **placeholder** solid-color
images shipped to satisfy the DMG layout config in `tauri.conf.json`.
Replace with brand artwork when available — keep dimensions at 660×400
(@1x) and 1320×800 (@2x) so the layout coordinates remain valid.

`README.txt` is bundled at the root of the mounted DMG via
`bundle.resources` in `tauri.conf.json`.
