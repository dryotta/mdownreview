A fast desktop app for browsing, reviewing and annotating markdown, code, and text files. Leave inline comments. Let your AI agent fix them.

How It Works

Review your AI agent's output in five steps — using Copilot skills that automate the entire loop.

1 Your AI agent writes code

Ask your coding agent to build a feature, fix a bug, or refactor. It produces files — you need to review them.

Copilot CLI
$ Build the auth module with JWT tokens Creating src/auth/validate.ts... Creating src/auth/refresh.ts... Creating src/auth/middleware.ts... ✓ Created 3 files

2 Open in mdownreview

Use the open skill to launch the app with your project folder. Browse the file tree, read rendered markdown, and leave inline review comments.

Files

📄 refresh.ts
📄 middleware.ts 1

Missing null check — token could be undefined when called from middleware
4 return verify(token, SECRET);

3 Summarize with read

The read skill scans all .review.yaml sidecar files and gives your agent a summary of every unresolved comment.

Copilot CLI
$ mdownreview:read 3 unresolved comments in 2 files: src/auth/validate.ts (2 comments) issue L3: Missing null check — token could be undefined suggestion L4: Use try/catch around verify() src/auth/middleware.ts (1 comment) issue L12: No rate limiting on token refresh

4 Fix with review

The review skill orchestrates the full cycle — it reads the comments, makes fixes, and marks them resolved. Your agent responds to every issue.

Files
📄 validate.ts
📄 refresh.ts
📄 middleware.ts
validate.ts
3export function validate(token: string | undefined) {
You issue
Missing null check — token could be undefined
Agent reply
Added union type and early return guard at L4. Marked resolved. ✓
4 if (!token) return { valid: false, error: 'no token' };
5 try { return { valid: true, payload: verify(token, SECRET) }; }

5 Clean up with cleanup

Once all comments are resolved, the cleanup skill removes the .review.yaml sidecar files, leaving your project clean.

Copilot CLI
$ mdownreview:cleanup Scanning for resolved review files... ✓ Removed validate.ts.review.yaml (2/2 resolved) ✓ Removed middleware.ts.review.yaml (1/1 resolved) Done — 2 sidecar files cleaned up

Agent Skills

Install the Copilot skills to let your AI agent read and act on review comments.

/plugin marketplace add dryotta/mdownreview-skills
/plugin install mdownreview@mdownreview-skills
SkillDescription
openFind, install, and launch the mdownreview desktop app
readScan for review sidecars and list unresolved comments
reviewOrchestrate the full cycle — read, fix, and clean up
cleanupDelete sidecar files where all comments are resolved

Updating

Keep the app and skills up to date.

🔄 App

mdownreview checks for updates automatically on launch and installs them in the background. No action needed.

🔌 Skills

Update the Copilot skills to get the latest features:

/plugin update mdownreview-skills
